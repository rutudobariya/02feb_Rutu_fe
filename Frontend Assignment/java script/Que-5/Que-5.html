What are undeclared and undefined variables ?



Ans:

Undefined :-

--> It occurs when a variable has been declared but has not been assigned any value.

--> Undefined is not a keyword .

Example:-

var apple ;
console.log(apple);

Output:-

Undefined




Undeclared :-

--> An Example of an undeclared variable is when there is no such variable in the program .

--> For Example, let's try to print a variable called without having such a variable in the program :

Example:-

console.log(banana);

Output:-

Uncaught ReferenceError : banana is not defined
