What is the function of the delete operator ?



Ans:

-->The javascript delete operator is used to remove an element from an array, objects, and properties from an object.

-->javascript delete operator is not a method; it's a statement.

Syntax:-



Syntax of Delete an object.

delete objectName

Syntax to delete the property of an object.

delete objectName.propertyName
