What is === Operator ?



Ans:

-->This Operator is Known as Strict equality . Whether its teo operands are equal, returning a Boolean result.
and Operands value and datatype both are same at that time it will generated true(Boolean result).


JavaScript Demo: Expressions - Strict Equality operator



console.log(1 === 1);
//Expected Output: true (both value are same and datatype(number)is also same)

console.log("Dhevan" === "Dhevan");
//Expected Output: true (both value are same and datatype(String)is also same)

console.log('5' === 5);
//Expected Output: False (both value are same and datatype is diffrent that's why result will be false )

console.log(0 === false);
//Expected Output: False (both value is diffrent and one value type is number and another is string that's why result will be false)
