What is negative infinity?



Ans:

NEGATIVE_INFINITY is a special numeric value that is returned when an arithmetic operation or mathematical function generates a negative value greater than the largest representable number in JavaScript.

Number.NEGATIVE_INFINITY returns negative Infinity

Number.NEGATIVE_INFINITY is " a number lower then any other number ".

Negative Infinity is diffrent from mathematical infinity the following ways :

  1. Negative Infinity results in -0 When divided by any Number .
  2. When Divided by itself Positive Infinity ,Negative Infinity is NAN
  3. Negative Infinity when divided by any positive number is Negative Infinity .

Syntax:-

let n = (-Number.MAX_VALUE) * 2;
