Q-18 What are the media types allowed by CSS?

-->One of the most popular features of style sheet whi is used to present the document on different media.
types of media Query:

  1. All (includes all media types)
  2. screen (used for computer screens, tablets, smart-phones etc,)
  3. prient(used for printers)
  4. Speech (used for screen readers that"reads" the page out loud)
-->media queries in css3 extended the css2 media types idea.
