Q-16 What are the advantages and disadvantages of using external style sheets?

    the Advantages of External style sheet are as follows:

  • with the help of External style sheets the style of numerous documents can be oraganized from one single file.
  • in External style sheet, classes can be made for use on numberous html elements type in many forms of the site.
  • in complex contexts,methods like selector and grouping can be implemented to apply style .

    the Advantages of External style sheet are as follows:

  • An extra download is essenial to import style information for each file.
  • the execution of the file may be deffered till the external sheet is loaded.
  • while implementing style sheets,we need to test web pages with multiple browsers in order to check comatiblity issues.
