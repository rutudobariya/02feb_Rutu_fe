    Q-12 How to create a Hyperlink in HTML?

    --> HTML links are hyperlinks.Select the text or picture that you want to display as a hyperlink.You can click on a link and jump to another document.use the anchor tag.The target attribute specifies where to open the linked document.

    1. _self:Default. Opens the document in the same window/tab as it was clicked
    2. _parent:Opens the document in a new window or tab
    3. _black: Opens the document in the parent frame
    4. _Top: Opens the document in the full body of the window
    Example:
    < a href="https://www.flipkart.com" target="_self">Flipkart< /a>
    < a href="https://www.flipkart.com" target="_parent">Flipkart< /a>
